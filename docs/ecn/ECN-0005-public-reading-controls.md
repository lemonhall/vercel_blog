# ECN-0005: 前台阅读控制与管理动作入口

## 基本信息

- **ECN 编号**：ECN-0005
- **关联 PRD**：PRD-0001
- **关联 Req ID**：REQ-0001-004、REQ-0001-005
- **发现阶段**：v3 前台基础体验修整
- **日期**：2026-06-10

## 变更原因

真实迁移完成后，公开文章已经能显示，但当前前台列表存在基础使用缺口：没有分页，不能按时间正序/逆序切换，宽屏下正文可用宽度不足，站点缺少 footer 和更明确的柠檬品牌识别。同时，前台文章列表和详情页缺少面向单人管理员的编辑/删除入口。

## 变更内容

### 原设计

PRD-0001 的公开阅读体验只要求首页列表、详情页、搜索、移动端稳定和基础可读样式；后台能力只要求创建、编辑、保存富文本文章和图片上传。

### 新设计

- 站点品牌标题前增加柠檬 emoji，favicon 使用柠檬标识。
- 首页增加分页，并支持按发布时间正序/逆序切换。
- 首页和文章详情页支持宽模式，减少宽屏两侧留白。
- 首页每篇文章右下角显示编辑/删除动作。
- 文章详情页末尾右下角显示编辑/删除动作。
- 删除必须是逻辑删除：服务端只把 `posts.status` 更新为 `deleted`，不得物理删除数据库行。
- 未登录管理员访问编辑/删除动作时跳转到后台登录；已登录时执行对应功能。
- 全站增加 footer，形成明确页面收尾。
